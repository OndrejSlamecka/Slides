Slides—HTML & CSS based presentations

A HTML5 & CSS3 presentations boilerplate
by Ondrej Slamecka

← Control by arrows or clicking on the background →

Usage

  • Download from GitHub (git clone git://github.com/OndrejSlamecka/Slides.git)
  • Edit HTML

Want better look? Edit CSS, change background (Subtle Patterns?)

  • Slides are under BSD-3 license. Make sure you are ok with that.

Uses

  • jQuery
  • jQuery doubletap plugin (I haven't tested this on touch device yet – if you did, please email me)
  • A background pattern from Subtle Patterns
